# やさしすぎ家計簿

「今日これだけ入力すればOK」と思えるくらい、ゆるく続けられる家計簿アプリです。

## できること

- ログイン / 新規登録
- 初回のみの予算設定
- 質問形式の記録フロー
- データページでカテゴリ残高の確認
- カレンダー表示で日・週・月の集計
- 設定画面で月予算やカテゴリの変更
- レシート画像の読み取り補助
- 記録データの変更 / 削除

## 画面

- ログイン画面
- 初回設定
- ホームのチャット画面
- データページ
- カレンダーページ
- 設定画面

## 起動方法

このリポジトリは HTML / CSS / JavaScript の静的アプリです。

1. `index.html` をブラウザで開きます
2. もしくは GitHub Pages で公開します

## 画像素材

不足するキャラクター素材は `assets/mascot/` に保存しています。

- `mikeneko-normal.png`
- `mikeneko-happy.png`

## 補足

- 端末内で `localStorage` に保存します
- 将来のクラウド同期に備えて、状態は `app.js` でひとまとめに管理しています
- レシートのAI読み取りはブラウザ側の `Tesseract.js` を利用します。環境によっては手入力での運用になります
